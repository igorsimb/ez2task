Creating a Company

Why would I want to create a company?
Creating a company allows you to create new users and collaborate with them on your projects
What changes?
You will have access to 2 new pages:
  • Manage Users
  • Manage Tasks
How do I create a company?
Once you click the "Create a company" link, you will be able to give it a name and a description (optional)
How do I add a new user?
  • Go to Manage Users page
  • Click 'Add User' button
  • Fill out the form with new credentials, they will be automatically added to your company
  • Done! New user can now log in using these credential
Note: we strongly encourage the new user to change their password following the "Reset password" link in user profile settings.
Do all users have the same privileges?
No. There are 2 account types:
  • Admin. Has access to all company's information and activities
  • Agent. Has access mostly to their own information and tasks only
Note: Admin can change other users' account type at any time by going to their profile page
Can I change the name of or delete my company?
You can change your company's name by following the 'Update company's info' link in your profile settings. You can't delete your company.
